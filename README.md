# Business Calculus Workbook

This repository contains the LaTeX source files for a Business Calculus workbook
developed using Overleaf.

## Purpose
The source files are provided to support accessibility, transparency, and
alternative format generation (e.g., HTML or EPUB).

## Build
The workbook can be compiled using LaTeX (PDFLaTeX or XeLaTeX).
A custom class file is included in the repository.
